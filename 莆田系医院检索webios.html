    莆田系医院检索[Web/iOS]

    5/10/2016

    [dy]莆田系，特指中国大陆的一个民营医药体系，是福建莆田民营医疗从业者的总称，其资本和主要人员背景主要来自福建省莆田市东庄镇。2014 年，全世界范围内的莆田籍从事医疗行业的企业家共同发起成立了莆田（中国）健康产业总会。据媒体报道，莆田系掌握着中国大陆 80% 的民营医疗份额。维基百科 (opens new window)[/dy]

    这里介绍了三种方式来查询医院是否属于莆田系，青小蛙对医疗行业不是很了解，只希望提供方便让大家了解到某些医院的背景情况。

    注意这里并不一定是全部的莆田系医院，所以，请谨慎。

    # Open Power Hospital (opens new window)[Web]

    本项目由 Open Power 小组共同维护。可以基于地理位置检索。 本项目的第一批医院名单来源于凤凰网医院名单 (opens new window)，后根据社区热心人的贡献汇总而成。进入这份名单的医院符合以下三个条件中的至少一个：

    • 曾经投放百度竞价广告
    • 曾经被媒体报道过医疗欺诈事件
    • 与莆田系存在直接关系

    项目位于 Github，你可以参与协作或使用数据在你的项目中。

    # 莆田系医院查询 (opens new window)[Web]

    这是一个网页查询工具，输入医院名称就可以筛选菜是否属于莆田系医院，并能显示所属公司及法人。 作者提供了莆田系医院核查流程 (opens new window)，并强调了“一条一条人工审核”。同样可以提交以及 api 支持。 目前有 1118 条数据

    # 莆田医院检索 (opens new window)[iOS]

    数据来源于网络，作者未明示(这点很不好，用了别人的东西怎么能不署名呢？)，收录了全国各地 700 多家疑似莆田系医院，可以通过地名或医院名检索。 鉴于查询医院并不属于常规事情，推荐使用前两者快速查询(手机上均可打开)，并没有必要安装一个应用。

    [ps]第一个在线我这上不去，web搜索可以用省会名和市名分别搜索两次[/ps]
