# 二 SSH

putty连接即可，注意，进入以后输入密码时，是不显示的，光标也不动，但实际上是输入了,第一次登陆会提示你一个信息框，点**是(Yes)**就行了。 有网友做了服务器端一键安装ShadowSocks的脚本很方便！ 用PUTTY登录你的VPS，复制安装脚本，右键粘贴，然后点回车执行，按照提示一步步傻瓜式操作分分钟就可用搞定了。 在此把DEBIAN,UBUNTU,CENTOS下SS的一键安装包都整理分享一下： Debian下shadowsocks-libev一键安装脚本 by tennfy 下载脚本

wget --no-check-certificate https://raw.githubusercontent.com/tennfy/shadowsocks-libev/master/debian_shadowsocks_tennfy.sh

更改属性

chmod a+x debian_shadowsocks_tennfy.sh

安装

bash debian_shadowsocks_tennfy.sh

卸载

bash debian_shadowsocks_tennfy.sh uninstall

更新

bash debian_shadowsocks_tennfy.sh update

shadowsocks的配置文件 /etc/shadowsocks-libev/config.json vi该文件从而修改密码、服务器端口及加密方式，修改之后记得保存重启。 重启 /etc/init.d/shadowsocks-libev stop /etc/init.d/shadowsocks-libev start Debian/Ubuntu下shadowsocks-libev一键安装脚本 by teddysun 下载

wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-libev-debian.sh

属性

chmod +x shadowsocks-libev-debian.sh

安装

./shadowsocks-libev-debian.sh 2>&1 tee shadowsocks-libev-debian.log

卸载

./shadowsocks-libev-debian.sh uninstall

启动：/etc/init.d/shadowsocks start 停止：/etc/init.d/shadowsocks stop 重启：/etc/init.d/shadowsocks restart 状态：/etc/init.d/shadowsocks status centOS下shadowsocks-libev一键安装脚本 by teddysun 下载

wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-libev.sh

属性

chmod +x shadowsocks-libev.sh

安装

./shadowsocks-libev.sh 2>&1 tee shadowsocks-libev.log

卸载

./shadowsocks-libev.sh uninstall

安装完成后即已后台启动 shadowsocks ，运行： ps -ef grep ss-server grep -v ps grep -v grep 可以查看进程是否存在。 本脚本安装完成后，会将 shadowsocks-libev 加入开机自启动。 使用命令： 启动：/etc/init.d/shadowsocks start 停止：/etc/init.d/shadowsocks stop 重启：/etc/init.d/shadowsocks restart 查看状态：/etc/init.d/shadowsocks status
