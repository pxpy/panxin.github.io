Windows Live Writer是一款由微软开发的强大的离线发布文章工具，也是我最喜欢的建站软件之一。 在我看来，使用Windows Live Writer发布文章的好处在于: 1.可以同时管理多个博客，不需要登录博客后台即可发文章 2.可以离线写文章，尤其是断网时可以编辑好文章，上网再发布 3.强大的编辑器，批量插入图片非常方便 ……

# Windows Live Writer 安装和配置

这里演示的是Windows Live Writer 2012，XP用户就自己折腾2009吧，大同小异。 1.下载 (opens new window),官方地址：http://windows.microsoft.com/zh-cn/windows-live/essentials (opens new window) 注意：最新版2012支持win7及以上系统，所以xp系统打开后看不到立即下载 (opens new window) (opens new window) 2.运行安装包，选择安装Writer，然后一步步即可安装完成 注意，下载的只是个安装程序，安装过程中需要联网下载真正的安装包。 (opens new window) (opens new window) 3.运行Writer，选择wordpress(根据你的情况，我用的是wordpress建的博客) 4.填写网址和你的账号密码 (opens new window) 5.选择日记类型为WordPress，这时的远程发布网址一般为：http://你的网址/xmlrpc.php，下一步 6.可以下载或不下载主题（有些复杂主题是没办法下载的），然后就完成配置了 7.再次启动，同意协议，你就可以正常使用Windows Live Writer发布文章啦！ 8.当然也有许多插件可以选择，默认弹出的网址是英文的，这里给出中文地址： http://windows.microsoft.com/zh-cn/windows/download-essentials-plugins (opens new window)
