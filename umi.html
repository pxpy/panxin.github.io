    # 简介

    Umi 官网：https://umijs.org/ GitHub：https://github.com/umijs/umi

    1. 蚂蚁的，基于react
    2. 免费企业级
    3. 建议学习乌米3
    4. 请求库用axios

    # 不能使用umi

    • IE8或者更低浏览器

    # 使用

    可能网络不好会失败yarn前一共不到4kb

    yarn create @umijs/umi-app
yarn
