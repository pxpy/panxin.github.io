    pe还是很常用的，可以检查系统，里面还有很多工具，包括常用的重装系统工具，也可以在本机系统不能开机时救急！！

    PE是Preinstallation Environment的简称，即预安装环境。详细定义可以参见百度百科，这里不妨将PE理解为Windows系统的“超级精简、超级权限版”。在PE下我们可以安装操作系统、系统修复、分区等操作，即便Windows系统出现故障无法进入时，仍可进入PE对硬盘上的数据进行处理。 本文提供的这个“微PE工具箱”是“通用PE工具箱”的原作者"UEPON"基于Win8内核制作，纯净无广告。可以将PE安装到当前系统下，开机通过启动菜单可以进入；也可以将PE安装到U盘、移动硬盘，当Windows系统无法进入时，可以通过U盘或移动硬盘启动进入PE；此外，它还可以将PE生成ISO格式的文件，方便的进行光盘刻录。 (opens new window) (opens new window) [box style="tips"] 下载地址：http://www.wepe.com.cn/download.html (opens new window)[/box] 使用说明： 新机器推荐64位，老机器推荐32位。如果出现XXX已停止工作等提示，请试着用右键兼容性模式打开。 操作系统支持： Windows 10/8.1/8/7/Vista/XP/2000、Windows Server 2003/2008/2011/2012 “微PE工具箱”v1.0特征： 1.微PE的智能EXE安装包使WinPE能够灵活地安装到系统开机启动项、制作可启动U盘、移动硬盘、生成ISO镜像。 2.全面并完美地支持各个操作系统，甚至是在最新的UEFI启动的系统。 3.让复杂的技术操作简单化，让维护效率迅速提升，安装PE从此变得特别简单。 4.对全部软件都进行了精简和压缩适配，在保证功能完整的情况下，减小了存储和传输的成本。 5.选了一大批备受好评的维护型工具软件，这些软件均来自经验丰富的开发者，每个都很好用。

    以前经常用的老毛桃，现在是越来越流氓了，体积也越来越大，用它安装系统直接被修改，默认安装360，微PE工具箱很好。
